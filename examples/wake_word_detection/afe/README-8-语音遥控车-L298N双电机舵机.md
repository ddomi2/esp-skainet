### 简化连线表

| ESP32 GPIO | 连接目标 | 功能 |
|:---:|------|------|
| 10 | L298N ENA | 左电机 PWM 调速 |
| 11 | L298N IN1 | 左电机方向 A |
| 12 | L298N IN2 | 左电机方向 B |
| 13 | L298N ENB | 右电机 PWM 调速 |
| 14 | L298N IN3 | 右电机方向 A |
| 15 | L298N IN4 | 右电机方向 B |
| 16 | 舵机信号线 | 转向角度 PWM |
| GND | L298N GND + 舵机 GND + 电池负极 | **必须共地！** |

> ⚠️ **重要：ESP32 和 L298N 必须共地 (GND 连在一起)，否则信号无法正确传递！**

## 3. 语音命令

使用方式：先说 **"嗨，乐鑫"**（唤醒词），然后说命令词。

| 语音 | 拼音 | 动作 | 效果 |
|------|------|------|------|
| **"前进"** | qian jin | ACT_CAR_FORWARD | 双轮正转 + 舵机回正 |
| **"后退"** | hou tui | ACT_CAR_BACKWARD | 双轮反转 + 舵机回正 |
| **"左转"** | zuo zhuan | ACT_CAR_LEFT | 左轮停/右轮转 + 舵机左打45° |
| **"右转"** | you zhuan | ACT_CAR_RIGHT | 左轮转/右轮停 + 舵机右打135° |
| **"停止"** | ting zhi | ACT_CAR_STOP | 双轮停 + 舵机回正 |
| **"停车"** | ting che | ACT_CAR_STOP | 同上（同义词） |

### 操作流程

```
1. 说 "嗨，乐鑫"     → 唤醒成功
2. 说 "前进"          → 小车向前跑
3. 说 "嗨，乐鑫"     → 再次唤醒
4. 说 "左转"          → 小车左转
5. 说 "嗨，乐鑫"     → 再次唤醒
6. 说 "停止"          → 小车停下
```

## 6. L298N 驱动原理

### H 桥电路

L298N 内部包含两个 H 桥，每个 H 桥可以独立控制一个电机的正反转。

```
         IN1=HIGH, IN2=LOW → 正转
  ┌──────────────────────────────┐
  │         VCC                  │
  │    ┌─S1─┐    ┌─S2─┐         │
  │    │    │    │    │         │
  │    └──┤ MOTOR ├──┘         │
  │       │       │             │
  │    ┌──┤       ├──┐         │
  │    │    │    │    │         │
  │    └─S3─┘    └─S4─┘         │
  │         GND                  │
  └──────────────────────────────┘

  IN1  IN2  │ 结果
  ─────────┼──────
  HIGH LOW  │ 正转
  LOW  HIGH │ 反转
  LOW  LOW  │ 停止（滑行）
  HIGH HIGH │ 刹车
```

### PWM 调速

ENA/ENB 引脚接收 PWM 信号，占空比决定电机转速：
- 0% → 电机不转
- 50% → 半速
- 100% → 全速

代码中使用 1kHz PWM（电机驱动常用频率），默认速度 70%。

### 关键代码

```c
/* 前进：两轮同时正转 */
void gpio_motor_move(MOTOR_FORWARD) {
    // IN1=HIGH, IN2=LOW → 左轮正转
    gpio_set_level(IN1, 1);
    gpio_set_level(IN2, 0);
    // IN3=HIGH, IN4=LOW → 右轮正转
    gpio_set_level(IN3, 1);
    gpio_set_level(IN4, 0);
    // ENA=70%, ENB=70% → 两轮同速
    update_pwm(70, 70);
}

/* 左转：左轮停，右轮正转 → 绕左轮轴心转向 */
void gpio_motor_move(MOTOR_LEFT) {
    gpio_set_level(IN1, 0);  // 左轮停
    gpio_set_level(IN2, 0);
    gpio_set_level(IN3, 1);  // 右轮转
    gpio_set_level(IN4, 0);
    update_pwm(0, 70);
}
```

## 7. 舵机 PWM 原理

### 标准舵机信号

舵机通过 PWM 脉宽来确定目标角度：

```
     20ms 周期 (50Hz)
  ├───────────────────────────────────┤

  ├─0.5ms─┤                              → 0° (最左)
  █████████░░░░░░░░░░░░░░░░░░░░░░░░░░░

  ├──1.5ms──┤                            → 90° (居中)
  █████████████████░░░░░░░░░░░░░░░░░░░░

  ├────2.5ms────┤                        → 180° (最右)
  █████████████████████████████░░░░░░░░░
```

### 角度映射

| 角度 | 脉宽 | 14位 duty 值 | 用途 |
|:---:|:---:|:---:|------|
| 0° | 0.5ms | 409 | 最左 |
| 45° | 1.0ms | 819 | 左转 |
| **90°** | **1.5ms** | **1228** | **居中（直行）** |
| 135° | 2.0ms | 1637 | 右转 |
| 180° | 2.5ms | 2047 | 最右 |

### 关键代码

```c
/* 角度 → duty 值的线性映射 */
static uint32_t angle_to_duty(int angle)
{
    // 0° → 409,  180° → 2047
    return 409 + (2047 - 409) * angle / 180;
}

/* 设置舵机角度 */
void gpio_servo_set_angle(int angle)
{
    uint32_t duty = angle_to_duty(angle);
    ledc_set_duty(mode, channel, duty);
    ledc_update_duty(mode, channel);
}
```

## 10. 常见问题

### Q: 电机不转？

1. **检查共地** — ESP32 GND 和 L298N GND 必须连在一起
2. **检查供电** — L298N 12V 端需要 7~12V 电池供电
3. **检查跳线帽** — L298N 板上 ENA/ENB 旁的跳线帽要拔掉（改用 ESP32 PWM 控制）
4. **检查接线** — IN1~IN4 是否接对 GPIO 编号

### Q: 电机转但不受控制（乱转）？

- 共地问题：ESP32 和 L298N 的 GND 没连
- 电源不足：电池电压太低，换满电电池

### Q: 舵机抖动？

- 供电不足：舵机需要较大电流（峰值 500mA+），建议单独 5V 供电
- 不要用 ESP32 的 3.3V 给舵机供电

### Q: 语音命令识别不到"前进""后退"等？

- 这些是自定义命令词（不在 313 条内置命令中）
- 确认 `custom_commands[]` 中已正确添加
- 确认 `action_map[]` 中拼音拼写正确

### Q: 能否调节速度？

可以！在 `cmd_handler.c` 中添加"加速"/"减速"命令：

```c
// 在 custom_commands 中添加:
{ 419, "jia su", "加速" },
{ 420, "jian su", "减速" },

// 在 action_map 中添加:
{ "jia su", "加速", ACT_CAR_SPEED_UP },
{ "jian su", "减速", ACT_CAR_SPEED_DOWN },

// 在 switch 中添加:
case ACT_CAR_SPEED_UP:
    gpio_motor_set_speed(gpio_motor_get_speed() + 10);
    break;
case ACT_CAR_SPEED_DOWN:
    gpio_motor_set_speed(gpio_motor_get_speed() - 10);
    break;
```
