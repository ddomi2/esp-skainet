## 1. 坦克 vs 小车

```
┌──────────────────────────────────────────────────────────┐
│          坦克模式（当前实现）      小车模式               │
│                                                          │
│    左履带    右履带          左轮       右轮              │
│   ┌─────┐  ┌─────┐       ┌───┐     ┌───┐              │
│   │ ◄►  │  │ ◄►  │       │   │     │   │              │
│   │     │  │     │       └─┬─┘     └─┬─┘              │
│   │     │  │     │         │  舵机   │                 │
│   │     │  │     │         └──┤├────┘                  │
│   └─────┘  └─────┘              │                       │
│                              前轮转向                    │
│   转向方式: 差速/反转         转向方式: 舵机偏转          │
│   左转=左反右正              左转=舵机左打               │
│   硬件: 只需 L298N           硬件: L298N + 舵机          │
│   引脚: GPIO 10~15 (6个)     引脚: GPIO 10~16 (7个)     │
└──────────────────────────────────────────────────────────┘
```

| 对比 | 坦克模式 | 小车模式 |
|------|---------|---------|
| 转向方式 | 两履带差速/反转 | 舵机偏转前轮 |
| **能原地旋转** | ✅ 是（左右轮反向） | ❌ 不能 |
| 转弯半径 | 可以为 0 | 取决于舵机角度 |
| 硬件 | 只要 L298N + 2电机 | L298N + 2电机 + 舵机 |
| 适合场景 | 履带车、麦克纳姆轮 | 普通四轮/三轮车 |

## 3. 接线图

```
┌──────────────── L298N 模块 ────────────────┐
│  12V ← 电池正极 (7.4~12V)                 │
│  GND ← 电池负极 ← ESP32 GND (共地!)       │
│  5V  → 可给 ESP32 供电                     │
│                                            │
│  ENA ← GPIO 10  (PWM, 左电机速度)          │
│  IN1 ← GPIO 11  (左电机方向A)              │
│  IN2 ← GPIO 12  (左电机方向B)              │
│  OUT1/OUT2 → 左电机                        │
│                                            │
│  ENB ← GPIO 13  (PWM, 右电机速度)          │
│  IN3 ← GPIO 14  (右电机方向A)              │
│  IN4 ← GPIO 15  (右电机方向B)              │
│  OUT3/OUT4 → 右电机                        │
└────────────────────────────────────────────┘

⚠️ L298N 板上 ENA/ENB 旁的跳线帽要拔掉！（改用 ESP32 PWM 控制速度）
⚠️ ESP32 GND 和 L298N GND 必须连在一起（共地）！
```

### 连线表

| ESP32 GPIO | → 连接 | 功能 |
|:---:|------|------|
| 10 | L298N ENA | 左电机 PWM 调速 |
| 11 | L298N IN1 | 左电机方向 A |
| 12 | L298N IN2 | 左电机方向 B |
| 13 | L298N ENB | 右电机 PWM 调速 |
| 14 | L298N IN3 | 右电机方向 A |
| 15 | L298N IN4 | 右电机方向 B |
| GND | L298N GND + 电池负极 | **共地** |

## 4. 语音命令

先说 **"嗨，乐鑫"**（唤醒），再说命令词：

| 语音 | 拼音 | 电机动作 |
|------|------|---------|
| **"前进"** | qian jin | 左轮 ↑ 右轮 ↑ (同向同速) |
| **"后退"** | hou tui | 左轮 ↓ 右轮 ↓ (同向同速) |
| **"左转"** | zuo zhuan | 左轮 ↓ 右轮 ↑ (原地旋转) |
| **"右转"** | you zhuan | 左轮 ↑ 右轮 ↓ (原地旋转) |
| **"停止"** | ting zhi | 左轮 ✕ 右轮 ✕ |
| **"停车"** | ting che | 同上 |
| **"加速"** | jia su | 速度 +10% (最大 100%) |
| **"减速"** | jian su | 速度 -10% (最小 0%) |

### 操作示例

```
"嗨，乐鑫" → "前进"    → 坦克向前开 (70%速度)
"嗨，乐鑫" → "加速"    → 速度升到 80%
"嗨，乐鑫" → "左转"    → 原地左旋转
"嗨，乐鑫" → "前进"    → 继续向前
"嗨，乐鑫" → "停止"    → 停下来
```

## 5. 坦克转向原理

### 核心思想：差速转向

坦克没有方向盘，靠两条履带的**速度差**和**方向差**来转向：

```
┌─────────────────────────────────────────────────────┐
│                     前进方向 ↑                       │
│                                                     │
│  ┌─── 前进 ───┐   ┌─── 后退 ───┐                   │
│  │ 左↑   右↑  │   │ 左↓   右↓  │                   │
│  │ ██↑   ↑██  │   │ ██↓   ↓██  │                   │
│  │ ██↑   ↑██  │   │ ██↓   ↓██  │                   │
│  └────────────┘   └────────────┘                    │
│                                                     │
│  ┌─ 原地左转 ─┐   ┌─ 原地右转 ─┐                   │
│  │ 左↓   右↑  │   │ 左↑   右↓  │                   │
│  │ ██↓   ↑██  │   │ ██↑   ↓██  │                   │
│  │ ██↓   ↑██  │   │ ██↑   ↓██  │                   │
│  │   ↺ 逆时针  │   │   ↻ 顺时针  │                   │
│  └────────────┘   └────────────┘                    │
│                                                     │
│  ┌─ 缓左转 ──┐   ┌─ 缓右转 ──┐                    │
│  │ 左慢  右快  │   │ 左快  右慢  │                    │
│  │ ██↑   ↑██  │   │ ██↑   ↑██  │                    │
│  │ (30%) (100%)│   │(100%) (30%) │                    │
│  │    ↰ 弧线  │   │    弧线 ↱  │                    │
│  └────────────┘   └────────────┘                    │
└─────────────────────────────────────────────────────┘
```

### 三种转向对比

| 转向类型 | 左电机 | 右电机 | 效果 | 代码 |
|---------|:---:|:---:|------|------|
| **原地左转** | 反转 ↓ | 正转 ↑ | 原地逆时针旋转，转弯半径=0 | `MOTOR_LEFT` |
| **原地右转** | 正转 ↑ | 反转 ↓ | 原地顺时针旋转，转弯半径=0 | `MOTOR_RIGHT` |
| **缓左转** | 慢(30%) | 快(100%) | 弧线左转，向前+偏左 | `MOTOR_SOFT_LEFT` |
| **缓右转** | 快(100%) | 慢(30%) | 弧线右转，向前+偏右 | `MOTOR_SOFT_RIGHT` |

### 关键代码

```c
case MOTOR_LEFT:
    /* 坦克原地左转：左轮反转，右轮正转 → 原地逆时针旋转 */
    set_motor_direction(IN1, IN2, false);   // 左轮反转
    set_motor_direction(IN3, IN4, true);    // 右轮正转
    update_pwm(speed, speed);               // 两轮同速
    break;

case MOTOR_SOFT_LEFT:
    /* 缓左转：左轮慢速，右轮全速 → 弧线转弯 */
    set_motor_direction(IN1, IN2, true);    // 两轮都正转
    set_motor_direction(IN3, IN4, true);
    update_pwm(speed / 3, speed);           // 左慢右快
    break;
```

## 6. L298N 驱动原理

### IN1/IN2 方向控制真值表

| IN1 | IN2 | 电机状态 |
|:---:|:---:|---------|
| HIGH | LOW | 正转 (前进) |
| LOW | HIGH | 反转 (后退) |
| LOW | LOW | 停止 (滑行) |
| HIGH | HIGH | 刹车 |

### ENA/ENB PWM 调速

```
ENA 占空比:    0%        50%        100%
电机转速:     不转       半速        全速
              ░░░░░   █████░░░░   ██████████
```

代码使用 **1kHz PWM**（电机驱动常用频率），默认速度 **70%**。

### L298N 模块实物标注

```
         ┌──────────────────────────────┐
  OUT1 ──┤                              ├── OUT3
  OUT2 ──┤        L298N 模块            ├── OUT4
         │                              │
  12V ───┤  [跳线帽]    [跳线帽]        │
  GND ───┤   ↑ 拔掉!     ↑ 拔掉!       │
  5V  ───┤                              │
         │  ENA  IN1  IN2  IN3  IN4  ENB│
         └──┬────┬────┬────┬────┬────┬──┘
            │    │    │    │    │    │
           G10  G11  G12  G14  G15  G13
                    ESP32 GPIO
```

> ⚠️ **跳线帽必须拔掉**：默认跳线帽将 ENA/ENB 直接接到 5V（全速），
> 拔掉后改为 ESP32 的 PWM 信号控制速度。

## 10. 附录：小车模式（舵机）

如果你使用的是**有舵机的三轮/四轮小车**（前轮转向），只需修改两处：

### 1. cmd_handler.c — 加回舵机调用

```c
case ACT_CAR_FORWARD:
    gpio_motor_move(MOTOR_FORWARD);
    gpio_servo_center();            /* 方向盘回正 */
    break;

case ACT_CAR_LEFT:
    gpio_motor_move(MOTOR_SOFT_LEFT);   /* 用缓转代替原地旋转 */
    gpio_servo_set_angle(45);           /* 舵机左打 */
    break;

case ACT_CAR_RIGHT:
    gpio_motor_move(MOTOR_SOFT_RIGHT);
    gpio_servo_set_angle(135);          /* 舵机右打 */
    break;

case ACT_CAR_STOP:
    gpio_motor_move(MOTOR_STOP);
    gpio_servo_center();                /* 回正 */
    break;
```

### 2. 额外接线

```
GPIO 16 → 舵机信号线 (橙)
5V      → 舵机电源 (红，建议独立供电)
GND     → 舵机地 (棕)
```

## 常见问题

### Q: 坦克只有一边履带转？
- 检查另一侧 OUT 线是否接好
- 检查对应 IN 引脚的 GPIO 编号

### Q: 转向时坦克不是原地转而是走弧线？
- 确认代码用的是 `MOTOR_LEFT`（原地转）而不是 `MOTOR_SOFT_LEFT`（弧线转）
- 检查两侧电机速度是否一致

### Q: 电机不转？
1. **共地** — ESP32 GND ↔ L298N GND ↔ 电池负极 必须连通
2. **跳线帽** — ENA/ENB 旁的跳线帽已拔掉
3. **供电** — 12V 端有 7~12V 电池
4. **接线** — IN1~IN4 对应正确的 GPIO

### Q: "加速"/"减速" 没效果？
- 需要先执行 "前进" 或其他运动命令，停止状态下改速不可见
- 速度范围 0~100%，超出会自动截断
